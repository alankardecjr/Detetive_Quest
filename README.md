🕵️‍♂️ Jogo de Investigação Criminal: O Mistério da MansãoEste projeto é um jogo interativo de dedução e investigação criminal desenvolvido em C. O objetivo do jogador é explorar os cômodos de uma mansão para coletar pistas e, com base nelas, acusar o verdadeiro culpado.O projeto foi dividido em duas versões independentes para demonstrar a evolução de uma interface de terminal para uma interface gráfica:VsConsole: Versão clássica baseada em texto que roda direto no terminal.VsGrafico: Versão moderna com interface visual, botões interativos e renderização em tempo real utilizando a biblioteca Raylib.🛠️ Arquitetura e Estruturas de DadosO núcleo do jogo foi construído utilizando conceitos fundamentais de estruturas de dados avançadas para garantir eficiência e organização:Árvore Binária de Salas (O Mapa): Os cômodos da mansão são conectados como nós de uma árvore binária. Cada quarto possui ponteiros para seus filhos (esquerda e direita) e um ponteiro de retorno para o nó pai, permitindo uma navegação natural de ir e voltar pelos ambientes.Árvore Binária de Busca - BST (Inventário de Pistas): Todas as pistas coletadas pelo detetive são inseridas em uma BST. Isso garante que, ao abrir o inventário ou finalizar o jogo, as evidências sejam exibidas perfeitamente organizadas em ordem alfabética.Tabela Hash com Encadeamento (Mapeamento de Evidências): Para associar as pistas aos suspeitos de forma instantânea $O(1)$, utilizamos uma Tabela Hash com tratamento de colisão por lista encadeada. Ela valida se o suspeito acusado possui relação direta com as pistas coletadas.
